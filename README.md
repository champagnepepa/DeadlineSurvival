# DeadlineSurvival
Deadline Survival adalah game 3D First-Person Survival Horror yang dikembangkan menggunakan Unity Engine dan bahasa C#. Proyek ini berfokus pada implementasi pendekatan Finite State Machine (FSM) untuk mengatur kecerdasan buatan (Artificial Intelligence) pada perilaku musuh (Zombie), mencakup state: Idle, Patrol, Chase, Attack, dan Death.
